# AI tools worthy of interest

Short, curated notes on AI tools worth evaluating.

## Agent runtimes and orchestration

- **Superserve** — An open-source Firecracker microVM runtime for durable, isolated agent sessions; supports stateful pause/resume, snapshots, workspace forks, and harnesses such as Codex, Claude Code, OpenCode, and Pi. [superserve.ai](https://www.superserve.ai/)
- **Paseo** — A self-hosted, open-source interface for running and orchestrating coding agents—including Codex, Claude Code, OpenCode, and Pi—from desktop, web, mobile, or CLI while retaining native provider setups. [paseo.sh](https://paseo.sh/)
